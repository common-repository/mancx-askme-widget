The AskMe widget by Mancx is the ultimate monetizing tool for bloggers and experts.

== Description ==

AskMe enables you to answer questions for money on your WordPress blog or website. It provides an excellent opportunity to engage your visitors in real business.

Mancx offer a very simple business model, where you answer the questions and we supply the platform, handle all taxes and administration. You keep 80% of the revenues you generate.

Your visitors think highly of you, and are willing to pay for your knowledge. Do not let them down!

All questions generated by the AskMe widget are managed on a specific user account here at mancx.com. The widget is specific for you, and questions asked will not be published elsewhere.

== Installation ==

This section describes how to install the plugin and get it working.

1. Search for the mancx askme widget plugin in the WordPress plugin repository in your blog's admin interface (http://blog.example.com/wp-admin/plugin-install.php?tab=search&type=term&s=mancx&plugin-search-input=Search+Plugins)
2. Click the 'Install Now' link. You will be prompted if you really want to install the plugin -- click 'OK'.
3. When the plugin has finished installing, activate the plugin by clicking 'Activate Plugin' link.

== Alternate Installation ==

1. Download the 'mancx-askme-weidget.zip' archive.
2. Upload the 'mancx-askme-weidget.zip' archive through the install widget upload page in your blog's admin interface (http://blog.example.com/wp-admin/plugin-install.php?tab=upload).
3. When the plugin has finished installing, activate the plugin by clicking 'Activate Plugin' link.

== Usage ==

1. Create a new Mancx account or connect an existing one in order to use the Mancx AskMe Widget on your blog. (http://blog.example.com/wp-admin/admin.php?page=mancx)
2. Drag the Mancx AskMe widget to one of your blog's widget areas using the Appearence -> Widgets page (http://blog.example.com/wp-admin/widgets.php)
3. Customize the widget to your liking and click 'Save.'
4. Done! Now visit your blog to see view the widget.

== Frequently Asked Questions ==

= Do I need a Mancx account to start using the Mancx AskMe widget? =

No! You will need an account, but you can create one using the plugin after you have installed it!
